# 前言

今天看到[思否](https://segmentfault.com)有关于远程办公经验分享的征文活动，而我们公司从成立之初就采用了远程工作这种形式，所以我想借着这个机会，谈一谈自己这一年来的远程工作体验，希望能够对想要尝试远程办公的朋友们有所帮助。

# 入职

新义互联是一个年轻的移动互联网广告公司，刚刚成立两年，我是在去年愚人节入职的。

从入职前的面试开始就全部是远程进行的。远程面试是一对一的，事先通过邮件或电话约好时间和面试方式，然后在约好的时间准时开始面试，面试通过约定好的方式进行，或电话，或微信视频，或QQ视频等，总之，面试的方式很灵活。

我入职时，公司规模还比较小，仅经过了四轮面试就通过了，前三轮是技术面试，第四轮是老板亲自面试。

当然签入职合同以及岗前培训还是需要到北京总部跑一趟的。

我们的业务部门都在北京总部，业务部门的同事当时还是需要每天去公司上班的，我签入职合同时，北京总部只有三个同事，现在已经壮大成十几个人的队伍了。

因为最近的新型冠状病毒肺炎问题，我们从春节假期之前开始，公司的业务人员也都开始在家远程办公了。

而我们公司的技术开发人员从一开始就是分布在全国各地进行远程工作的，现在还有在国外的。

# 每天日常

每天早上差不多 7 点左右起床做饭，7 点半吃早饭，8 点之前步行送孩子上学，9 点钟之前回到家，然后进书房开始工作，中午 12 点左右开始做饭吃饭，吃完之后，接着进书房工作，下午 5 点步行接孩子放学，到家之后做饭吃饭，吃完之后，继续进书房工作，9 点之后，洗漱睡觉。

上面当然只是理想情况，因为家里有三个孩子，老大平时是上学的，老二还没到上学的年龄，我们没有送他去幼儿园，他在家每天自己玩，倒是也挺省心，老三比较小，去年的时候还不到一岁，还好我老婆平时也在家，平时基本上都是她来照顾老三，不过给孩子擦屁股、换纸尿裤、洗澡之类的活，还是需要我帮忙的。所以，即使在工作时间，如果需要我帮忙照顾孩子的话，我还是会离开椅子一会儿的，所以倒也不用担心长时间久坐得痔疮啦。

中午吃饭时间其实也没那么准，有时候灵感来了，写代码停不下来，等想起吃饭的时候，可能已经下午 2 - 3 点了。而晚上有时候孩子睡得早，夜深人静正好可以专心写代码，一不小心可能也会写到半夜 12 点之后。不过最近已经尽量减少熬夜行为了，身体健康才是最重要的。

另外，老大放寒暑假时，因为不用接送上学了，上午 8 点左右就可以坐在电脑前开始工作了，下午也可以一直工作到吃晚饭时间，虽然实际工作时间长了，但是感觉比接送孩子还要轻松。

至于锻炼身体的话，对我个人来说，平时孩子上学时，我步行接送孩子就算最大的锻炼了。之前家里有一个Xbox360，带 Kinect 体感的，但是现在住的地方房间有点小，不适合用它玩体感运动游戏，就送人了。后来买个 WiiU，又买了个 WiiU 的平衡板，有时候早上起的早，会玩一玩上面的体感运动游戏，就算是锻炼身体了。不过说实话，我平时坚持的不好，一个星期能通过它运动两次已经算多的啦。希望以后自己能在这方面多多努力吧，不知道买个 Switch 的健身环大冒险会不会让自己锻炼能够坚持的好一些😂。

# 每周日常

通常每个周一早上 11 点和每个周五下午 6 点我们都会准时开一个全体视频会议，周一的会议是确定这一周业务上要做的工作，周五的会议是总结一下这一周的业务工作情况，每次会议通常都控制在半个小时左右。

我们技术组，是每两周算一个 SPRINT，第一周的周一早上 10 点，会准时开一个 SPRINT 任务确认会议，第二周的周五下午 4 点会准时开一个 SPRINT 任务总结会议。我们技术组的会议通常都很短，每次会议基本上平均也就 10 分钟左右。

# 每季度日常

公司每个季度会制定一次 OKR，并总结上一季度 OKR 的完成情况。在制定 OKR 时，由公司先制定总体的 OKR，然后根据总体的 OKR，各部门再制定各部门的 OKR，最后每个人根据各自部门的 OKR 来制定自己的 OKR。另外，每年的年底还会制定一次新年度的 OKR 和做一次本年度的 OKR 总结。

OKR 不以考核为目标，OKR 是用来确定一个共同的奋斗目标，大家针对这个共同的奋斗目标来制定各自明确的量化的任务，然后朝着这个目标去努力完成。但是我们知道有时计划没有变化快，尤其是在这个瞬息万变的年代，我们在工作中，常常会发现某个开始制定的任务，因为某些原因已经没有必要做了，或者没有那么急迫的需要去完成它，取而代之的是，有更急迫但是在一开始并没有在计划中的任务等着去做。这种时候，我们通常会以最终奋斗目标为导向去调整工作中的任务，因此，OKR 可以更好的发挥每个人的自主性和创造性，使自由和方向达成一种平衡。

公司每个季度除了 OKR 的制定和总结外，还会举行一次团建活动，团建活动一般是一周左右。因为平时大家都在家远程办公，很少有机会见面，团建活动就成了大家见面交流，促进感情的一个很好的机会，有时候 OKR 的制定和总结也会在团建中进行，这样可以更好的面对面交流。团建活动我们一般会选一些比较好玩的地方，最近两次团建的地点是黄山和泰国普吉岛。

# 远程工作所用到的工具

我们在远程工作中，主要用到了下面一些工具：

## 1. Github

我们所有的代码都是放在 Github 上的，有私有项目，也有开源项目。为了方便国内用户访问，部分开源项目我们还在 Gitee 上做了镜像。我们的 Code Review 都是在 Github 上完成的，我们的一部分开发文档，也是放在 Github 项目的 Wiki 中的。Github 可以算是我们项目开发的基础平台了。

## 2. Slack

我们技术组的日常交流、项目部署，系统监控等等工作都是通过 Slack 这个工具完成的。Slack 提供了很多集成工具，比如某个同事在 Github 上提交了代码，在 Slack 上就可以收到需要 Review 的消息，Jenkins 同时也会把提交代码的构建状态报告到 Slack 里面，项目部署时，在 Slack 里面直接敲入部署命令，机器人就会进行部署，并把部署状态反馈在 Slack 中，所以，每个人当前在做什么，做了什么，在 Slack 里面都可以清楚的看到，并且 Slack 还保留了所有记录，所以要查找之前的某个操作也很方便。另外，我们的系统如果出现异常情况，也会把报警信息发送到 Slack 里面。有了它，使我们的远程工作变得非常方便。

## 3. Confluence

我们每次会议的会议记录，OKR 的制定和总结，内部文档的管理，个人的分享等等内容，我们是记录在 Confluence 中的，它提供了很好的分类管理的功能，另外，还可以多人共同编辑文档，这一点让远程办公多人合作编辑文档变得非常方便。不过缺点是它不支持 md 格式的文档编辑，因此，一些用 md 格式编写的开发文档我们放在了 Github 上，然后在 Confluence 中仅作了一个指向 Github 中的文档的链接。

## 4. 微信

我们跟业务部门以及跟合作伙伴之间是通过微信进行沟通的。之前我们开视频或语音会议时，也用过它，不过后来改用钉钉了。

## 5. 钉钉

签到，报销，请假管理我们用的都是钉钉。在前期，没用钉钉的时候，我们是没有签到的，报销和请假直接微信上发个消息就行了。但是随着人员的增多，这种方式就不方便统计了，所以就引入了钉钉，除了使用钉钉的这些管理功能以外，我们把视频会议也改在钉钉上开了。另外，我们的服务都是放在阿里云上的，因此，使用钉钉也方便我们跟阿里云的同志们进行沟通。不过钉钉的很多功能只有手机端才有，电脑端没有，需要在手机上才能操作，我手机可能用的时间太久了，现在非常的慢，导致使用起来非常浪费时间。

## 6. Teambition

SPRINT 任务管理我们使用的就是它。

## 7. 邮件

邮件也是必不可少的，我们系统很多功能都跟邮件是集成的。

另外，我们也用过 Zoom，Telegram，虽然很不错，但是因为环境原因，这些工具在国内香不起来，只能放弃。

# 远程工作感受

其实从 2008 年开始，我就已经开始在家工作了，最初也是像现在这样的远程工作，但那个时候没有这么多方便的工具，主要是靠 Skype 和邮件来进行远程协同工作。再后来就一直自己在家工作，做一些开源项目什么的。所以在加入现在的公司之后，并没有感觉在工作方式有什么不适应的地方。

至于时间管理上，我不用番茄钟之类的工具，虽然曾经尝试过，但是感觉不适合我。一方面因为它会屏蔽消息，这样就会导致我不能及时看到同事们发的消息，不能做出及时的响应，另一方面它会影响我写代码的专注力，比如我代码刚写到兴头上，它突然提醒我该休息了，就把我思路打断了。

我之前自己在家工作的时候，是通过电脑上的便签来管理自己每天要做的任务，每天打开电脑把今天想做的事情一条一条写到便签上，每完成一件就删掉一行。

现在远程工作时，任务都是写在 teambition 里面的，每天工作之前，打开 teambition 来找到自己今天要做的任务，标记上进行中，完成之后，将标记改为已完成，并标记上完成用了多少时间，这样每完成一个任务就像在游戏中打败BOSS通过一关一样，让我感觉很有挑战性，也很有成就感，所以感觉这种方式比用番茄钟之类的高效多了。

我们公司平时都是双休，周末有两天的时间可以陪孩子们玩，不过因为家里孩子多，老大和老二周末他们会自己联机玩游戏，所以实际上也很少需要我陪，通常我会在周末做自己的开源项目。

在家办公还有个好处，就是不用整天背着笔记本往返于公司，所以我就可以在家配一台性能强劲的台式机，再配上两个显示器，其中主显示器是 43 寸的 4K 显示器，写代码非常爽，副显示器是 23 寸的 1080p 显示器，用来放 Slack 和微信聊天窗口，这样就可以及时的看到同事们发的消息了。有了这样的设备之后，工作效率那是相当的高啊。

所以这种长期在家工作的状态我感觉其实挺好的，并没有觉得有什么孤独寂寞感，而且能够很好的平衡工作和家庭之间的关系，既不耽误工作，又能照顾好家，也不用把时间浪费在上下班的路上，感觉真是棒极了。

# 总结

总之，感觉自己能够加入我们公司真是一件非常幸运的事，毕竟现在国内像我们这样全部技术开发人员都可以在家工作，没有 996，拥有双休，每个季度有一次国内团建旅游，每年还有一次国外团建旅游的公司真是太少了，而且同事们都非常的棒，能够一起工作真是太开心了。
